## ğŸ”§ Development Guide

### **Adding a New Module**
1. Create `newmodule.html` in root
2. Add link to navigation in `main.js`
3. Create any needed CSS/JS files
4. Update the dashboard module cards

### **Modifying Styles**
- **Global variables:** `css/themes.css`
- **Component styles:** `css/main.css`
- **Responsive design:** `css/responsive.css`

### **Adding New Data**
1. Add JSON to `data/` directory
2. Load with `fetch()` in respective JS file
3. Update UI to display new data

### **Key JavaScript Modules**

| Module | Purpose | Key Functions |
|--------|---------|---------------|
| `main.js` | Core app logic | Navigation, modals, animations |
| `auth-simulator.js` | User management | Sign up, login, session handling |

| `chatbot.js` | AI assistant | Conversation flows, quick replies |
| `map-simulator.js` | Metro map | Station data, SVG rendering |
| `store-builder.js` | Store builder | Drag & drop, layout saving |
